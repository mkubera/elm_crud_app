# elm_crud_app
 CRUD App - Elm - Jexia

## How to run
1. Create your own `.env` file (check `.dummy_env` to get an idea 😄)
1. `npm i`
1. `npm start`
1. Open your ❤️ browser, and navigate to `localhost:3000`
